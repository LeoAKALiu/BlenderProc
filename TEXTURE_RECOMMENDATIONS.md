## 🎯 真实光伏施工场地的特征

根据真实场景分析，光伏施工场地通常包含：

1. **多种土壤类型**
   - 红土/红壤（red soil）
   - 黄土/黄壤（yellow soil）
   - 深色土壤（dark brown soil）
   - 浅色/沙质土壤（light/sandy soil）

2. **植被覆盖**
   - 干草/枯草（dry grass）
   - 稀疏草地（sparse grass）
   - 裸露土壤区域（bare soil）

3. **施工痕迹**
   - 推土机推过的区域（bulldozed/scraped areas）
   - 车辙痕迹（tire tracks）
   - 压实土壤（compacted soil）

4. **地形特征**
   - 不平整的地面（uneven terrain）
   - 台阶/梯田（terraces）
   - 道路/通道（roads/pathways）

## ✅ 建议补充的贴图类型

### 1. **土壤类型贴图** (优先级：高)

#### 1.1 红土/红壤 (Red Soil)
- **用途**：模拟中国南方常见的红壤
- **特征**：红褐色、偏红
- **建议命名**：`RedSoil001_4K-JPG` 或 `SoilRed_4K-JPG`
- **必需文件**：
  - `*_Color.jpg` - 红褐色基础颜色
  - `*_NormalGL.jpg` - 法线贴图
  - `*_Roughness.jpg` - 高粗糙度（0.8-0.9）
  - `*_Displacement.jpg` - 轻微位移

#### 1.2 黄土/黄壤 (Yellow Soil)
- **用途**：模拟中国北方常见的黄土
- **特征**：黄褐色、偏黄
- **建议命名**：`YellowSoil001_4K-JPG` 或 `SoilYellow_4K-JPG`
- **必需文件**：同上

#### 1.3 深色土壤 (Dark Brown Soil)
- **用途**：模拟湿润、有机质丰富的土壤
- **特征**：深褐色、偏黑
- **建议命名**：`DarkSoil001_4K-JPG` 或 `SoilDark_4K-JPG`
- **必需文件**：同上

#### 1.4 浅色/沙质土壤 (Light/Sandy Soil)
- **用途**：模拟干燥、沙质土壤
- **特征**：浅褐色、偏灰白
- **建议命名**：`SandySoil001_4K-JPG` 或 `SoilSandy_4K-JPG`
- **必需文件**：同上

### 2. **植被贴图** (优先级：高)

#### 2.1 干草/枯草 (Dry Grass)
- **用途**：模拟施工场地周围的干枯植被
- **特征**：黄褐色、枯黄色、有纹理
- **建议命名**：`DryGrass001_4K-JPG` 或 `GrassDry_4K-JPG`
- **必需文件**：
  - `*_Color.jpg` - 枯黄色基础颜色
  - `*_NormalGL.jpg` - 法线贴图（模拟草叶纹理）
  - `*_Roughness.jpg` - 中等粗糙度（0.6-0.8）
  - `*_Displacement.jpg` - 轻微位移

#### 2.2 稀疏草地 (Sparse Grass)
- **用途**：模拟部分覆盖的草地
- **特征**：绿色与土壤混合
- **建议命名**：`SparseGrass001_4K-JPG`
- **必需文件**：同上

### 3. **施工痕迹贴图** (优先级：中)

#### 3.1 推土机推过的区域 (Bulldozed/Scraped Soil)
- **用途**：模拟桩基行下方的推土痕迹
- **特征**：平整、有方向性纹理、颜色较浅
- **建议命名**：`BulldozedSoil001_4K-JPG` 或 `ScrapedSoil_4K-JPG`
- **必需文件**：
  - `*_Color.jpg` - 浅色、有方向性纹理
  - `*_NormalGL.jpg` - 法线贴图（模拟推土痕迹）

#### 3.2 压实土壤 (Compacted Soil)
- **用途**：模拟被车辆压实的区域
- **特征**：平整、光滑、颜色较深
- **建议命名**：`CompactedSoil001_4K-JPG`
- **必需文件**：同上

## 🔧 代码改进建议

### 当前代码的贴图加载逻辑
代码在 `load_terrain_textures()` 函数中搜索：
- `*red*soil*` 或 `*soil*red*` - 红土
- `*grass*` 或 `*dry*` - 干草
- `*tire*` 或 `*track*` - 车辙

### 建议的改进
1. **自动识别多种土壤类型**
   - 搜索 `*red*soil*`, `*yellow*soil*`, `*dark*soil*`, `*sandy*soil*`
   - 随机选择或按区域分配

2. **智能混合策略**
   - 桩基行下方：使用推土机推过的区域贴图
   - 道路区域：使用道路贴图
   - 其他区域：混合草地和多种土壤

3. **贴图命名规范**
   - 建议统一命名格式：`[Type][Number]_4K-JPG`
   - 例如：`RedSoil001_4K-JPG`, `DryGrass001_4K-JPG`

## 🎨 使用建议

### 贴图混合策略
1. **基础层**：使用主要土壤类型（红土或黄土）
2. **混合层**：使用噪声纹理混合草地和土壤
3. **细节层**：添加车辙、推土痕迹等细节
4. **变化层**：使用多种土壤颜色增加变化

### 参数建议
- **纹理缩放**：5.0-10.0（根据场景大小调整）
- **混合强度**：0.3-0.6（根据真实度需求调整）
- **粗糙度**：0.8-0.95（地面通常很粗糙）
- **位移强度**：0.2-0.5（轻微起伏）
